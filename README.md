## What’s new in PanoramiX v2.0.0

PanoramiX still focuses on one thing first:
capturing any scrollable Windows area as one continuous image.

This version adds a complete built-in editor:

- Precise cropping
- Real pixelation for sensitive information
- Transparent highlighting
- Freehand drawing in multiple colors
- Arrows, rectangles, circles and ✓ / ✕ markers
- Text annotations
- Continuous single-page PDF export

The application is still in beta and is not digitally signed yet.
Windows may display a warning during installation:

More info → Run anyway
